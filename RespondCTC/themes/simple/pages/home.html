Welcome to Respond!

This is the starter theme for Respond. Respond is a responsive content management system built in Angular that you can use to build responsive websites. Use it as a starting point to create your own site.

Better with Bootstrap

Respond is built on Twitter Bootstrap 3.0. This means the sites you create using Respond are built on a proven, powerful framework that is easy to extend and adapt to your needs. Building your site has never been easier with Respond and Bootstrap.

Make Money with Respond

Respond has built-in support for subscription billing using Stripe and a full featured e-commerce cart built-in to your sites. So whether you want to make money selling websites, or build a site to sell a product or service, Respond has you covered.

Beautiful Themes

Respond ships with a beautiful default theme and built-in support for the beautiful Bootswatch theme library - just change a line of CSS and go. Stunning premium themes are also now available for purchase in the Respond Marketplace.
